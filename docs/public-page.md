# DFフォームガード

DFフォームガードは、a-blog cms のフォーム送信をAIで判定し、営業メールや迷惑メールらしい送信による管理者通知の負担を減らす拡張アプリです。

## 主な機能

- フォーム送信内容をAIで判定し、問い合わせとして扱うべきか確認します。
- 不要な送信と判断した場合、管理者宛メールだけを抑制できます。
- フォーム履歴は通常どおり残し、あとから判定結果を確認できます。
- フォームごとに利用有無や追加判定ルールを設定できます。
- AI判定前に動く、軽量なHoneypot bot対策も利用できます。

## こんな場面に向いています

問い合わせフォームに営業メールや迷惑メールが多く届き、管理者の確認負担が大きくなっているサイトに向いています。正規問い合わせを残すことを優先しながら、不要な通知を減らすための補助ツールです。

## 利用について

AI判定を利用する場合はOpenAI APIキーを設定します。フォーム送信そのものを完全に止めるのではなく、フォーム履歴を残しながら管理者宛メールの扱いを調整する設計です。
